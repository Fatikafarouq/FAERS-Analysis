# FAERS Analysis
FAERS Q2 2025 reports for side effects of ibuprofen and analysis of other drugs for pharmacovigilance analysis 
This repository contains a dataset of 48900 reported side effects of ibuprofen extracted from the FDA Adverse Event Reporting System (FAERS) and also further anaktsis on other reported drugs
It is intended for use in pharmacovigilance, pharmacoepidemiology and drug safety.
The repository contains only side effects at this stage, value counts and outcome will be added in later updates

## Goal
The aim of this project is to 
Identify the most reported side effect of ibuprofen and understand the consequences with other drugs
Support ongoing research into NSAID safety profiles

## Future improvements
Add other NSAIDs to the list for comparison
Create visualizations

## About
Learning data analysis by working with real world pharmacovigigilance and epidemiology data
